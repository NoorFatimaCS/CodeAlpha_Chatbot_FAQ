# 🤖 AI FAQ Chatbot - CodeAlpha Internship Task

This is a professional AI-powered Chatbot designed to answer Frequently Asked Questions (FAQs) regarding the CodeAlpha internship. It uses Natural Language Processing (NLP) to understand user queries and provide accurate responses.

## 🚀 Live Demo
You can interact with the live chatbot here: 
👉 https://huggingface.co/spaces/noorfatimaimran/CodeAlpha-FAQ-Chatbot

## ✨ Features
- **NLP Powered:** Uses TF-IDF and Cosine Similarity for intelligent matching.
- **Modern UI:** Built with a professional, glassmorphism-inspired web interface.
- **Real-time Interaction:** Fast and responsive chat experience.
- **Cloud Hosted:** Deployed on Hugging Face Spaces using Docker.

## 🛠️ Tech Stack
- **Language:** Python 3.9
- **Framework:** Flask
- **Machine Learning:** Scikit-learn (TfidfVectorizer)
- **Frontend:** HTML5, CSS3 (Modern UI)
- **Deployment:** Docker & Hugging Face Spaces
